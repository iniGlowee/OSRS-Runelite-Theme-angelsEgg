<h3>Theme Installation:</h3>

1. Download the theme from my github
2. Open runelite
3. Navigate to the plugins section, search for 'Resource Packs'
4. Click the settings icon for this plugin
5. Update the dropdown labeled, 'Use Resource Pack' to be `First`
6. Under 'Resource Pack Paths', update the text input box for the labeled input element, 'Resource pack path 1' to be `C:\path\to\your\OSRS-Runelite-Theme-angelsEgg`


<h3>Theme Uninstallation:</h3>

1. Delete & remove my theme pack from your local files
2. Clear out the Resource pack path 1
3. Update the dropdown labeled, 'Use Resource Pack' to be `Hub`


FAQ:

1. Can I change this theme if I want?

    Yes just remember to give credit to melkypie & resource pack, but create your own repo don't try to update mine


2. The theme is not working? I cannot see the customizations

    Try putting the runlite `Game client layout` = `Resizable - Classic Layout`


3. I want to remove something, can you help me?

    No, search through the files and identify the part of the theme you want to remove or change & update the file
